capy whispers

Six prompt-engineering tips that move the needle

01

Front-load the camera

Open with the shot type, lens and lighting. img2 anchors composition first, subject second.

02

Quote the text you want

Wrap any in-image copy in "…". Quotes prevent the model from rephrasing it.

03

Pick the ratio first

Specify 16:9 / 4:5 / 9:16 at the top of the prompt — composition follows.

04

Reference > adjective

"In the style of Wes Anderson" is fine. Attaching a frame is better. img2 reads references like a moodboard.

05

Lock the seed for variations

Re-use the same seed and only swap one variable (outfit, lighting) to build true series.

06

End with negatives, sparingly

One short "Avoid: …" line beats five. Long negative blocks dilute the positive prompt.

FAQ

What is GPT Image 2?

GPT Image 2 (often shortened to img2) is OpenAI's latest image-generation model. It accepts long, structured prompts, multiple aspect ratios, and reference images, and produces sharper text rendering and stronger compositional control than previous generations.

Where do these prompts come from?

Each prompt is hand-picked, re-written for clarity, re-tested with GPT Image 2, and tagged with category, difficulty and aspect ratio. The full dataset lives in data/prompts.json.

Can I use the prompts commercially?

Prompt text is generally not copyrightable. The output you generate, however, is governed by your image-model provider's terms (OpenAI, Google, Byteplus, etc.). Check those before commercial use.

How do I add my own prompt?

The dataset is just data/prompts.json. Open a PR or drop a new entry — the gallery renders straight from JSON, so anything you add appears immediately.

What is the Capy chat panel?

Happycapy is the AI agent that powers this page. The right-side drawer opens a live Capy chat with your selected prompt pre-loaded, so you can generate, iterate or ask Capy to rewrite the prompt for a different style.

Got a prompt that should be here?
